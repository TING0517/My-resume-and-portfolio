碩士大數據輿情分析-ETtoday新聞雲輿情分析

HTML JavaScript Python BeautifulSoup CKIP Transformers Pandas Django

專案概述

從ETtoday 新聞雲爬取新聞，每個類別每天爬取約 15 篇新聞報導，共計 8 天，總共獲得了 358 篇新聞報導，並進行熱門關鍵字、熱門人物、熱門關鍵詞分析。
範例為2024年3月27日至2024年4月2日的新聞關鍵字分析結果，可以選擇不同的新聞類別或選擇“全部”以查看所有熱門關鍵字。可以選擇要匯出的關鍵字數量。輸出結果將另外以長條圖呈現。
