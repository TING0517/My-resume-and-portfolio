碩士大數據輿情分析-ETtoday新聞雲輿情分析

HTML JavaScript Python BeautifulSoup CKIP Transformers Pandas Django

專案概述

利用 Python 撰寫自動化爬蟲程式，針對 ETtoday 新聞雲 進行多類別數據採集。專案期間（如 2024/03/27 - 04/02）每日定額擷取各類別新聞，累積 358 篇 原始文本。透過自然語言處理技術實施熱門關鍵字、核心人物及主題分析，並整合視覺化工具產出動態長條圖，協助使用者快速洞察跨領域的社會輿論焦點。
範例為2024年3月27日至2024年4月2日的新聞關鍵字分析結果，可以選擇不同的新聞類別或選擇“全部”以查看所有熱門關鍵字。可以選擇要匯出的關鍵字數量。
